# Team Stats

## Description

This project uses JavaScript objects and arrays to store and manage sports team statistics.

## Features

* Store player information
* Store game results
* Retrieve players and games using getters
* Add new players
* Add new games

## Concepts Used

* Objects
* Arrays
* Getters
* Methods
* push()

## How to Run

1. Save the code as `app.js`
2. Open terminal
3. Run:

```bash
node app.js
```
